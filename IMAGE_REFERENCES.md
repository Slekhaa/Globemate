## How to Replace an Image

1. Find the country/place in this file
2. Go to [Unsplash](https://unsplash.com) and find a replacement image
3. Copy the image URL (add `?w=1200&h=400&fit=crop` for backgrounds, `?w=400&h=300&fit=crop` for places)
4. Open `js/country-info.js`
5. Search for the old URL and replace it with the new one
6. Update this file to keep it in sync

**Tip:** Unsplash image URLs follow the pattern:  
`https://images.unsplash.com/photo-{ID}?w={width}&h={height}&fit=crop`
